Subtopic: System Security

Member Name: Rey Vaca

1. Introduction

  • Lack of security causes digital systems to be volatile and unsafe to browse.
  • To ensure the safety of their users, developers use different measures to protect against threats to a digital system.

2. What is System Security? (Source: Florida Tech, 2024; GeeksforGeeks, 2022)

  • System security is defined as the precautions an organization takes to ensure its networks and resources are safe from downtime, interference or malicious intrusion.
  • The goal of a digital security system is to provide integrity, secrecy, and availability to all its users.
  • It is in the best interest of an organization to secure its digital systems so they work properly and are not accessed by unwanted individuals.

3. Threats to Digital Systems (Sources: GeeksforGeeks, 2022; Impreva, 2024)

  • Program Threats – Program threats include viruses, trojan horses, ransomware, etc. These programs were written by unautorized users to perform malicious or extortive acts. If successful, an attacker can gain ahold of an organizations entire system
  • Social threats – This threat involves social engineering. An attacker will convince a victim to install malware in order to gain sensitve information and then go on to act as someone else (ususally for money).
  • Denial of Service (Dos) – This attack sends the system a large volume of traffic, slowing all functions to a halt. Multiple devices sending these attacks are called Distributed DoS (DDoS).

4. Security Measures Taken (Sources: GeeksforGeeks, 2022; Impreva, 2024)

  • Network Security – All of the information the is vital to an organizations system runs through its network. Securing these network beforehand can help prevent harmful breaches in data.
  • Cloud Security – secure cloud storage can be a strong precaution against unwated attackers. Good cloud storage involves detecting and fixing false security configurations and vulnerbilities.
  • User Security – An organization's digital system is only as secure as its user's knowledge. Better education about threats and vulnerbilites can help ensure no user falls prey to social engineering attacks.

5. Conclusion

  • System security is crucial for an organization's digital system. Without it, valuable information they posses is at risk of being seized by unwated cyberattackers.
  • Proper security measures, constant maintenance, and user education will help prevent attacks.
