Subtopic: Unauthorized Use

Member Name: Lani Storm

1. Introduction

  • Unauthorized use refers to actions like jailbreaking devices or bypassing intended restrictions to gain unauthorized control or access.
  • To stay safe, users should avoid jailbreaking, create strong passwords, and avoid using public Wi-Fi networks.

2. Jailbreaking (Source: Malwarebytes, 2023)

  • Jailbreaking removes built-in security restrictions on devices to gain root access.
  • Although it allows more customization, it makes devices more vulnerable to hackers.
  • Using encryption and security tools can help protect jailbroken devices, but the best protection is to avoid jailbreaking altogether.

3. Importance of Strong Passwords (Source: Walden University)

  • Strong Password Tips: Use uppercase, lowercase, numbers, and symbols.
  • Minimum Length: Passwords should be longer than 8 characters to resist brute-force attacks.

4. Using Public Wi-Fi (Source: TechSafety Australia)

  • Risks: Public Wi-Fi networks can expose users to interception and tracking.
  • Safe Practices: Avoid public Wi-Fi or use a trusted VPN when accessing sensitive information.

5. Conclusion

  • Protecting personal data requires avoiding unnecessary risks like jailbreaking and using insecure networks.
  • Three key strategies: don’t jailbreak, use strong passwords, and avoid public Wi-Fi.
